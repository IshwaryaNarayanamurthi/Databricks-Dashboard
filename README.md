 Project Overview
 
This project delivers a comprehensive Retail Analytics Dashboard built using Databricks SQL. It provides end-to-end insights across sales, customers, products, payments, inventory, and store operations.
The dashboard is designed to support data-driven decision-making by combining KPI tracking, trend analysis, and operational monitoring
